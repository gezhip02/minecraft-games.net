About Mine Clone

Mine Clone is a cool Minecraft inspired game in which you can use your creativity to build whatever you can imagine! You can interact with this blocky world and build different blocks – create structures, cities – anything!

In this game, you can also complete different challenges and gain daily rewards. Furthermore, you can also interact with the various animals and creatures on the map. See what you can build and explore this blocky world to the max!

Game Features:

  • Daily tasks to complete
  • A big blocky world to explore
  • Animals and creatures around the map
  • Creativity-style gameplay
  • Save and Load functionality
  • Web browser platform

How to Play:

  • WASD or arrow keys to move
  • Mouse scroll up to switch weapons
  • Left mouse button to attack
  • E to open the inventory
  • Right mouse button to place an item
  • Space bar to jump
  • F to open the task menu
  • 1234 to use accessory
  • Esc to open the menu

Mine Clone was originally released in April 2013 and was updated in August 2019. It was further updated in February 2021 with a working Save and Load functionality! The game was developed by Girts Kesteris (Sacristan).
