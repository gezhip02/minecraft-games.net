About Pixel Warfare

Pixel Warfare is an IO multiplayer shooting game featuring blocky graphics and 8-bit music, offering a retro experience. The fast-paced, tactical gameplay delivers an exciting FPS experience, with various weapons and maps that keep you on your toes. Choose a game from the menu screen and go straight into the action.

With its pixelated aesthetics and thrilling multiplayer battles, Pixel Warfare brings the excitement of first-person shooters to your browser without the need for downloads or installations. Jump into active game rooms or create your own customized battles with friends.

How to Play Pixel Warfare

Start an online battle instantly

Pixel Warfare features active game rooms where you can join and instantly start fighting in online battles! Each game room has several maps and various settings so that you can find the perfect game.

Use your arsenal of weaponry

Pixel Warfare features a variety of weapons you can scroll through when you've started a match. There's a shotgun, sniper, rocket launcher, machinegun, and many other weapons. They all contain a set amount of ammunition. You have every weapon by default, so there is no need to unlock them.

Create a private room

If you want full control over the game, you can start a personalized game room with custom options. Here you can choose the game mode, map, round duration, max players, weapons to allow, and more.

Build up your stats

Pixel Warfare will remember your kills, deaths, and K/D ratio, displaying them to you in the menu screen, so you can see how well you're doing over time.

Game Controls:

  • Left mouse button = action
  • Right mouse button = aim
  • WASD = move
  • Shift + W = run
  • Space = jump
  • C = crouch
  • Z = prone
  • Enter = chat
  • TAB = open menu/score list
  • 1 - 8 (keys) or mouse scroll wheel = weapon select
